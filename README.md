# GTA VI - Landing Page

A modern, interactive landing page for GTA VI built with React, featuring smooth animations and a responsive design.

## Features

- **Interactive Hero Section**: Engaging introduction with animated elements
- **Video Sections**: Showcasing game footage with scroll-triggered animations
- **Character Sections**: Dedicated sections for Jason and Lucia characters
- **Postcard Section**: Interactive postcard showcase
- **Responsive Design**: Optimized for all device sizes
- **GSAP Animations**: Smooth scroll-triggered animations throughout the site

## Technologies Used

- **React** - Frontend framework
- **GSAP** - Animation library with ScrollTrigger
- **Tailwind CSS** - Utility-first CSS framework
- **Vite** - Build tool and development server
- **React Responsive** - Responsive utilities for React

## Installation

### 1. Clone the repository:

```bash
git clone https://github.com/md-zeon/GTA-6-Landing-Page.git
```

### 2. Navigate to the project directory:

```bash
cd gta-landing
```

### 3. Install dependencies:

```bash
npm install
```

## Usage

To start the development server:

```bash
npm run dev
```

To build for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```

To run linting:

```bash
npm run lint
```
